# Крестики-нолики
Консольное приложение — классическая игра «Крестики-нолики» для двух игроков. Позволяет поочерёдно делать ходы, проверяет победителя и определяет ничью.

## Состав команды
* **Осмачко Диана** (модуль A — создание и вывод поля)
* **Малахова Алина** (модули B, C и main — логика игры, ходы и проверка победы)

## Функциональные возможности
* Создание игрового поля — поле 3x3 для игры
* Ход игрока — ввод координат и размещение X или O
* Проверка победы — определение победителя по строкам, столбцам и диагоналям
* Проверка ничьи — определение окончания игры без победителя
* Смена игроков — автоматическое переключение X и O

## Разработка и тестирование

В проекте реализована автоматическая проверка качества кода и логики игры.

### 1. Установка зависимостей
Для работы линтеров и тестов необходимо установить пакеты:
```bash
pip install pytest flake8 black
### 2. Запуск тестов
Для запуска автоматических юнит-тестов используйте команду:
python -m pytest tests/ -v
3. Проверка стиля кода (Linter)
Проверка соответствия стандартам PEP 8 с помощью Flake8:
python -m flake8 .
4. Форматирование кода (Formatter)
Автоматическое исправление оформления кода с помощью Black:
python -m black .
Инструкция по запуску
1.	Клонировать репозиторий.
2.	(Опционально) Создать виртуальное окружение: python -m venv venv.
3.	Запустить программу:
python main.py
